# Kamva UI Kit

## Installation

```
$ npm install --save-dev @kamva/ui-kit
```

## buttons

### usage

example:

```jsx
import React from "react";
import { Button, PrimaryButton } from "@kamva/ui-kit";

const Example = () => {
  return (
      <Button>primary</Button>
      <PrimaryButton secondary>secondary button</Button>
      <PrimaryButton round>?</Button>
  ) ;
}
```
